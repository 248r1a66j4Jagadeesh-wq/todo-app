# React Todo App

A simple Todo application built using React.

## Features

- Add Tasks
- Delete Tasks
- Mark Tasks Completed
- Search Tasks
- Clear Completed Tasks
- Local Storage
- Responsive Design

## Technologies Used

- React
- JavaScript
- CSS
- Vite

## Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/todo-app.git
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npm run dev
```
